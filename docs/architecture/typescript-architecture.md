# TypeScript Architecture

## Objetivo

Centralizar la configuración del compilador TypeScript.

## Jerarquía

base
│
├── nextjs
└── react-library

## Responsabilidades

### base

Opciones comunes.

- strict
- target
- module
- declaration

### nextjs

Configuración específica para aplicaciones Next.js.

### react-library

Configuración para librerías React reutilizables.
