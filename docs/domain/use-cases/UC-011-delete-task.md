# UC-011 Delete Task

## Goal

Delete a task.

## Actor

User

## Preconditions

- The user is authenticated.

## Main Flow

1. The user requests to delete a task.
2. The system deletes the task.
3. The system informs the user that the task has been deleted.

## Alternative Flows

_None._

## Postconditions

- The task no longer exists.

## Business Rules

_No additional business rules._
