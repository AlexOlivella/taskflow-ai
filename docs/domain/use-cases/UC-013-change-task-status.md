# UC-013 Change Task Status

## Goal

Change a task's status.

## Actor

User

## Preconditions

- The user is authenticated.

## Main Flow

1. The user requests to change a task's status.
2. The system changes the task's status.
3. The system informs the user that the task's status has been changed.

## Alternative Flows

_None._

## Postconditions

- The task's status is changed.

## Business Rules

_No additional business rules._
