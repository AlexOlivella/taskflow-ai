# UC-009 Create Task

## Goal

Create a task.

## Actor

User

## Preconditions

- The user is authenticated.

## Main Flow

1. The user requests to create a task.
2. The system creates the task.
3. The system informs the user that the task has been created.

## Alternative Flows

_None._

## Postconditions

- A new task exists.

## Business Rules

- A task must have a name.
