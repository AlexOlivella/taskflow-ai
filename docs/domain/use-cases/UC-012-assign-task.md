# UC-012 Assign Task

## Goal

Assign a task.

## Actor

User

## Preconditions

- The user is authenticated.

## Main Flow

1. The user requests to assign a task.
2. The system assigns the task.
3. The system informs the user that the task has been assigned.

## Alternative Flows

_None._

## Postconditions

- The task is assigned.

## Business Rules

- A task can only be assigned to a workspace member.
