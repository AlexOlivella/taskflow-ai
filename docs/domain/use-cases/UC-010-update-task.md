# UC-010 Update Task

## Goal

Update a task.

## Actor

User

## Preconditions

- The user is authenticated.

## Main Flow

1. The user requests to update a task.
2. The system updates the task.
3. The system informs the user that the task has been updated.

## Alternative Flows

_None._

## Postconditions

- The task is updated.

## Business Rules

- A task must have a name.
